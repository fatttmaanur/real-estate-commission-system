# Frontend

Nuxt frontend for the technical case dashboard.

## Run

```bash
npm install
npm run dev
```

The app runs on:

- `http://localhost:3001`

## Backend Connection

The frontend is configured to call:

- `http://localhost:3000`

Make sure the backend is also running before opening the frontend.

## Features

- Transaction create form
- Transaction edit flow
- Transaction delete action
- Breakdown panel
- Agent earnings lookup
- Search and stage filtering
